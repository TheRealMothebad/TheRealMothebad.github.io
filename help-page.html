In this web browser you will see a title at the top of the screen that will show you the title of the website that you searched or if no title is found will show the URL.

To navigate to different websites you will need to enter a URL in the address bar located just below the title, if that URL is invalid the HTML viewer will show an error message.

To be taken to the web page viewed before the current page click the back button, if you would like to move forward through previously viewed web pages click the forward button. Those buttons can be found at the top left of the web browser.

The center of the program will show you the website that you searched for. It is called HTML viewer, you can click on the links shown and they will take you to the appropriate website.

At the bottom of the web browser there will be a status bar that shows the URL of anything that you hover over.
